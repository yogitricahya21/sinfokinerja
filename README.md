# SinfoKinerja - Sistem Informasi Kinerja Personel

SinfoKinerja adalah aplikasi berbasis web yang dirancang untuk manajemen data personel, monitoring kinerja, dan pengarsipan biodata secara digital.

* **Monitoring Admin**: Fitur lihat detail data diri personel secara lengkap menggunakan Modal.
* **Ekspor PDF**: Cetak laporan biodata personel resmi dalam format PDF.
* **Keamanan Akun**: Fitur ganti email dan password secara mandiri melalui pengaturan privasi.
